-- SUMMARY --

Integrates the Dropkick plugin with Drupal.

-- INSTALLATION --

  1. Download the Dropkick (http://jamielottering.github.com/DropKick/) jQuery plugin and extract the file under sites/all/libraries.
  2. Download and enable the module.
  3. Configure at Administer > Configuration > User interface > Dropkick

-- DEPENDENCIES --

  1. Libaries API (http://drupal.org/project/libraries)
